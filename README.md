🌐 Decentralized Identity Platform



A web-based application that uses \*\*IPFS, Filecoin, and Web3.Storage\*\* to store and manage user documents in a decentralized way.



\---



📌 Overview



This project demonstrates how users can upload and access their files without relying on centralized servers.



⚠️ Note: This is a prototype and not a complete decentralized identity (DID) system.



\---



🚀 Features



\* Upload files to decentralized storage (IPFS)

\* Store data using Filecoin via Web3.Storage

\* Access files using CID (Content Identifier)

\* Simple frontend using HTML, CSS, JavaScript

\* Basic login using browser localStorage



\---



🏗️ Tech Stack



\* HTML, CSS, JavaScript

\* IPFS

\* Filecoin

\* Web3.Storage



\---



⚙️ How It Works



1\. User logs in (stored in browser localStorage)

2\. Files are uploaded using Web3.Storage API

3\. Files are stored on IPFS

4\. A CID is generated for each file

5\. Files can be accessed using the CID
